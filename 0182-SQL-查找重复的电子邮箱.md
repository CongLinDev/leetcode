# SQL-查找重复的电子邮箱

## 题目描述

编写一个 SQL 查询，查找 Person 表中所有重复的电子邮箱。

```
+----+---------+
| Id | Email   |
+----+---------+
| 1  | a@b.com |
| 2  | c@d.com |
| 3  | a@b.com |
+----+---------+
```

根据以上输入，你的查询应返回以下结果：

```
+---------+
| Email   |
+---------+
| a@b.com |
+---------+
```

## SQL

```sql
select Email from Person group by Email having count(Id) > 1;
```
